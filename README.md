## View the components

The latest version of the components are hosted [on GitHub Pages](https://travi.github.io/admin.travi.org-components)

## Local Development

### Install dependencies

```
$ nvm install
$ npm install
$ bundle install
```

### View the components
```
$ npm start
```

### Run verification
```
$ npm test
```
